Sample project in Scala Language.

Contains following runnable scala classes:
1. Scala/Java implementation of Pi calaculation Using Akka Actor model.
2. Telephone Number Coder for coding any number into words using a dictionary of words.
3. Mathematical Complex number scala class to represent a CompleX Number(With Real and imaginary numbers).

How to run?
You can either use scalac or sbt.

install scala vaersion 2.11.XX
install sbt 

open sbt command.
type ~run
and execute any of the Option classes.
