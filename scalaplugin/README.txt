Prototype for scala eclipse plugin project.

Features :
1 Use Scala AST in assisting scala file editing and debugging.
2. Scala interactive console
3. Simple Build Tool (sbt) interactive console
	Parses sbt output and displays a list of errors
    Marks errors in the text editor 

How to run?
import the project into Eclise Ide version 4.3
Run as Eclipse Ide
